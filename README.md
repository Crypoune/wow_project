# WoW Character Tracker

## Project Overview

This is a personal learning project built to explore Python and web development by creating a mini website that tracks World of Warcraft characters using the Blizzard API.

The goal is to retrieve character data (profile, stats, gear, etc.) from the official Blizzard API and display it through a simple web interface.

---

## Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript (to be defined)
- **API:** Blizzard World of Warcraft API

---

## Learning Objectives

- Practice Python fundamentals
- Understand how to work with external APIs (OAuth, REST)
- Build a simple backend web application
- Learn basic frontend integration with a Python backend
- Apply clean project structure and Git best practices
